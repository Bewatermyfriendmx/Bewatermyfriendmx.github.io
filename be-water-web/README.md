## 🚀 Cómo Desplegar (Gratis)

### Opción 1: Vercel (Recomendado)
1. Ve a https://vercel.com
2. Registrate con GitHub o Email
3. Haz click en "New Project"
4. Sube la carpeta `be-water-web`
5. ¡Listo! Tu sitio estará publicado

### Opción 2: GitHub Pages
1. Ve a https://github.com
2. Crea un nuevo repositorio llamado `be-water-web`
3. Sube todos los archivos
4. En Settings > Pages, selecciona "Deploy from a branch"
5. Elige `main` como rama
6. ¡Tu sitio estará en: username.github.io/be-water-web

### Opción 3: Surge.sh
```bash
npm install -g surge
cd be-water-web
surge
```

### Opción 4: Infinityfree
1. Ve a https://www.infinityfree.net
2. Registrate
3. Crea un nuevo sitio
4. Sube los archivos via FTP
5. ¡Listo!

## 🎨 Customización

- **Colores:** Busca en el HTML `:root` para cambiar variables de color
- **Textos:** Edita el contenido en el `index.html`
- **Imágenes:** Reemplaza los archivos en la carpeta `img/`
- **Fuentes:** Google Fonts (Playfair Display, Montserrat)
